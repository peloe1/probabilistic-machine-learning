Problem 1: Deriving VB for a simple model (1/2)¶

Consider the variational Bayesian approximation for the example model from the lecture (see 'simple_vb_example.pdf' in the materials of lecture 6). Derive the VB update for the factor $ q(\tau)$ in the example and complete the code block 'Problem 1' in the template given below.

=== BEGIN MARK SCHEME ===

\begin{align*} \log q(\tau) &= E_{z,\theta}[\log p(\mathbf{x},\mathbf{z},\tau,\theta)] + C \\ &= \log p(\tau) + E_{\mathbf{z}}[\log p(\mathbf{z}|\tau)] + C \\ &= (\alpha_0-1)\log\tau + (\alpha_0-1)\log(1-\tau) + E_{\mathbf{z}}[\sum_{i=1}^N (z_{i2}\log\tau + z_{i1}\log(1-\tau)] + C \\ &= (\alpha_0 + \sum_{i=1}^NE_{\mathbf{z}}[z_{i2}] -1)\log\tau + (\alpha_0+\sum_{i=1}^N E_{\mathbf{z}}[z_{i1}] - 1)\log(1-\tau) + C\\ &= (\alpha_0 + N_2 - 1)\log\tau + (\alpha_0 + N_1 - 1)\log(1-\tau) + C \\ \Rightarrow \quad q(\tau) &= Beta(\tau | \alpha_0 + N_2, \alpha_0 + N_1), \end{align*}

where we have defined $N_k = \sum_{i=1}^N E[z_{ik}]$, and $C$ is an arbitrary constant (not depending on $\tau$) on each row.

=== END MARK SCHEME ===

I will be using the same notation as in the materials of the lecture and 'simple_vb_example.pdf'.

$\log q^*(\tau) = E_{z,\theta}[\log p(x,z,\tau,\theta)] = \log p(\tau) + E_z [\log p(z \ | \ \tau)] + \text{const}$

$\log p (z \ | \ \tau) = \sum_{i=1}^N z_{n2} \log \tau + z_{n1} \log (1-\tau)$

$\Rightarrow E_z [\log p (z \ | \ \tau)] = E_z [\sum_{i=1}^N z_{n2} \log \tau + z_{n1} \log (1-\tau)] = \sum_{i=1}^N E_z [z_{n2} \log \tau] + E_z [z_{n1} \log (1-\tau)]$

$E_z [\log p (z \ | \ \tau)] = \sum_{i=1}^N \log \tau E_z [z_{n2}] + \log (1-\tau) E_z [z_{n1}] = \sum_{i=1}^N r_{n2} \log \tau + r_{n1} \log (1-\tau) $

$E_z [\log p (z \ | \ \tau)] = N_2 \log \tau + N_1 \log (1-\tau)$

Now $\log p(\tau) = (\alpha_0 - 1) \log \tau + (\beta_0 - 1) \log (1-\tau) + \text{const}$ (log density of the $\text{Beta}(\alpha_0, \beta_0)$ distribution)

Now we can substitute these to the $\log q^*(\tau)$'s expression:

$\Rightarrow \log q^*(\tau) = (\alpha_0 - 1) \log \tau + (\beta_0 - 1) \log (1-\tau) + \text{const} + N_2 \log \tau + N_1 \log (1-\tau) + \text{const}$

$\log q^*(\tau) = ((\alpha_0 + N_2) - 1) \log \tau + ((\beta_0 + N_1) - 1) \log (1-\tau) + \text{const}$

Now we notice this is the log density of the $\text{Beta}(\alpha_0 + N_2, \beta_0 + N_1)$ distribution.

Exponentiating both sides and we get:

$q^*(\tau) = \text{const} \cdot \tau^{(\alpha_0 + N_2) - 1} (1-\tau)^{(\beta_0 + N_1) - 1} $

And indeed this is the $\text{Beta}(\alpha_0 + N_2, \beta_0 + N_1)$ distribution.

Problem 2: Deriving VB for a simple model (2/2)¶

As in Problem 1, consider the variational Bayesian approximation for the example model from the lecture (simple vb example.pdf). Now, derive the VB update for the factor $q(\theta)$ in the example and complete the code block 'Problem 2 in the template below.

=== BEGIN MARK SCHEME ===

\begin{align*} \log q(\theta) &= E_{\mathbf{z},\tau}[\log p(\mathbf{x},\mathbf{z},\tau,\theta)] + C\\ &= \log p(\theta) + E_{\mathbf{z}}[\log p(\mathbf{x}|\mathbf{z},\theta)] + C \\ &= \log \mathcal{N}(\theta|0,\beta_0^{-1}) + E_{\mathbf{z}}[\sum_{i=1}^N z_{i2} \log\mathcal{N}(x_i|\theta,1)] + C \\ &= -\frac{\beta_0}{2}\theta^2 - \frac{1}{2}\sum_{i=1}^N r_{i2}(x_i-\theta)^2 + C \\ &= -\frac{\beta_0}{2}\theta^2 - \frac{1}{2}\sum_{i=1}^N r_{i2} (x_i^2 - 2x_i\theta + \theta^2) + C \\ &= -\frac{1}{2}\theta^2(\beta_0+N_2) + \theta \underbrace{\sum_{i=1}^Nr_{i2}x_i}_{=N_2\bar{x}_2} + C \\ &= -\frac{1}{2}\theta^2(\beta_0+N_2) + \theta N_2 \bar{x}_2 + C \\ &= -\frac{\beta_0+N_2}{2}[\theta^2 - 2 \frac{N_2\bar{x}_2}{\beta_0+N_2}\theta] + C \\ &= -\frac{\beta_0+N_2}{2}[\theta - \frac{N_2\bar{x}_2}{\beta_0+N_2}]^2 + C \\ \Rightarrow \quad q(\theta) &= \mathcal{N}(\theta|\frac{N_2\bar{x}_2}{\beta_0+N_2}, (\beta_0+N_2)^{-1}), \end{align*}

where $\bar{x}_2 = \frac{1}{N_2}\sum_{i=1}^N r_{i2}x_i$ and $C$ is an arbitrary constant on each row.

=== END MARK SCHEME ===

$\log q^*(\theta) = E_{z,\tau}[\log p(x,z,\tau, \theta)] = E_{z,\tau}[\log p(\tau) + \log p(\theta) + \log p(z \ | \ \tau) + \log p(x \ | \ z, \theta)]$

$\log q^*(\theta) = \log p(\theta) + E_z[\log p(x \ | \ z, \theta)] + \text{const}$

Let's simplify $E_z[\log p(x \ | \ z, \theta)]$:

$E_z[\log p(x \ | \ z, \theta)] = E_z[\sum_{i=1}^N z_{n1} \log N(x_n | 0,1) + z_{n2} \log N(x_n | \theta, 1)] = \text{const} + \sum_{i=1}^N E_z[z_{n2}] \log N(x_n | \theta, 1)$ since $E_z[z_{n1}] \log N(x_n | 0,1)$ is a constant w.r.t. to $\theta$.

$E_z[\log p(x \ | \ z, \theta)] = \text{const} + \sum_{i=1}^N r_{n2} \log N(x_n | \theta, 1) = \text{const} -\frac{1}{2} \sum_{i=1}^N r_{n2} (x_n - \theta)^2$

$E_z[\log p(x \ | \ z, \theta)] = \text{const} -\frac{1}{2} \sum_{i=1}^N r_{n2} x_n^2 + \theta \sum_{i=1}^N r_{n2} x_n - \frac{1}{2} \sum_{i=1}^N r_{n2} \theta^2 = - \frac{1}{2} N_2 \theta^2 + N_2 \bar{x}_2 \theta + \text{const}$

Also recall that $\theta \sim N(0, \beta_0^{-1}) \Rightarrow \log p(\theta) = \log N(\theta | 0, \beta_0^{-1}) = -\frac{1}{2} \beta_0 \theta^2 + \text{const}$

Putting these together we get:

$\log q^*(\theta) = -\frac{1}{2} \beta_0 \theta^2 - \frac{1}{2} N_2 \theta^2 + N_2 \bar{x}_2 \theta + \text{const} = -\frac{1}{2} [(\beta_0 + N_2) \theta^2 -2 N_2 \bar{x}_2 \theta] + \text{const} = -\frac{1}{2(\beta_0 + N_2)^{-1}} [ \theta^2 -2 \frac{N_2 \bar{x}_2}{\beta_0 + N_2} \theta] + \text{const} $

$\log q^*(\theta) = -\frac{1}{2(\beta_0 + N_2)^{-1}} (\theta - \frac{N_2 \bar{x}_2}{\beta_0 + N_2})^2 + \text{const}$

Now we recognize that this is the log density of a normal distribution with parameters: $\beta_2 = \beta_0 + N_2$ and $m_2 = \frac{N_2 \bar{x}_2}{\beta_0 + N_2}$. Thus exponentiating both sides we get: $q^*(\theta) = N(\theta | m_2, \beta_2^{-1})$

kl_div = []

for num_samples in range(5, 5001, 100):
    # SIMULATE THE TRUE DATA SET

    lambda_true = 4   # precision
    mu_true = 2       # mean

    data_set = np.random.normal(mu_true, sigma_true, num_samples)

    # SPECIFY PRIORS

    # lambda is the precision parameter of the unknown Gaussian
    # and it is given a prior distribution Gamma(a0,b0),
    # (a0 is the 'shape' and b0 the 'rate')
    a0 = 0.01
    b0 = 0.01 # These correspond to a noninformative prior

    # mu is the mean parameter of the unknown Gaussian
    # and it is given a prior distribution that depends on
    # lambda: N(mu0, (beta0*lambda)^-1)
    mu0 = 0
    beta0 = 0.001 # Low precision corresponds to high variance

    # (This is the so-called Normal-Gamma(mu0, beta0, a0, b0)
    # prior distribution for mu and lambda)


    # LEARN THE POSTERIOR DISTRIBUTION

    # Due to conjugacy, the posterior distribution is also
    # Normal-Gamma(mu_n, beta_n, a_n, b_n)

    sample_mean = sum(data_set) / num_samples;
    sample_var = sum((data_set - sample_mean)**2) / num_samples;

    mu_n = (mu0 * beta0 + num_samples * sample_mean) / (beta0 + num_samples)

Problem 4: Variational approximation for a simple distribution¶

Consider a model with two binary random variables $x_1 and $x_2, defined by the distributions:

$p(x_1=0) = 0.4$

$p(x_2=0|x_1=0) = 0.5$,

$p(x_2=1|x_1=0) = 0.5$

$p(x_2=0|x_1=1) = 0.9$,

$p(x_2=1|x_1=1) = 0.1$

Find a fully factorized distribution $q(x_1, x_2) = q_1(x_1)q_2(x_2)$ that best approximates the joint $p(x_1, x_2)$, in the sense of minimizing $KL(p || q )$ .

Note: For normal variational inference, we would rather minimize $KL(q || p )$; recall that, in general, $KL(p || q ) \neq KL(q || p )$ (see Barber: Bayesian Reasoning and Machine Learning, ch. Figure 28.1 as well as Chapter 28.3.4 and 28.3.5, for the dramatically different solutions that can result by minimizing the different quantities, as well as commentary on their relative usefulness for approximate inference). Here, well minimize $KL(p || q )$ , as that is algebraically simpler.

=== BEGIN MARK SCHEME ===

Solution¶

Let us parametrize $q$ such that $q_1$ and $q_2$ have the Bernoulli parameters $a = q_1(x_1=1)$ and $b = q_2(x_2=1)$. Now we have the following joint distributions $p(x_1,x_2)$ and $q(x_1,x_2)$:

$p(x_1,x_2)$ $x_1=0$ $x_1=1$
$x_2=0$ $0.20$ $0.54$
$x_2=1$ $0.20$ $0.06$
$q(x_1,x_2)$ $x_1=0$ $x_1=1$
$x_2=0$ $(1-a)(1-b)$ $a(1-b)$
$x_2=1$ $(1-a)b$ $ab$

Then we write $\text{KL}(p|q)$, denoting by $C$ an arbitrary constant; and find the zeros of the partial derivatives w.r.t. $a$ and $b$: \begin{align*} \text{KL}(p|q) &= \sum_{\mathbf{x}} p(\mathbf{x}) \ln \frac{p(\mathbf{x})}{q(\mathbf{x})} = -\sum_{\mathbf{x}} p(\mathbf{x}) \ln \frac{q(\mathbf{x})}{p(\mathbf{x})} \\ &= -0.2 \ln \frac{(1-a)(1-b)}{0.2} - 0.54 \ln \frac{a(1-b)}{0.54} - 0.2 \ln \frac{(1-a)b}{0.2} - 0.06 \ln \frac{ab}{0.06} \\ &= C - 0.4 \ln (1-a) - 0.6 \ln (a) - 0.74 \ln (1-b) - 0.26 \ln (b) \\ \frac{\partial}{\partial a} \text{KL}(p|q) &= \frac{0.4}{1-a} - \frac{0.6}{a} = 0 \quad\quad \Rightarrow a = 0.6 \\ \frac{\partial}{\partial b}\text{KL}(p|q) &= \frac{0.74}{1-b} - \frac{0.26}{b} = 0 \quad\quad \Rightarrow b = 0.26 \end{align*}

=== END MARK SCHEME ===

So we are trying to find that $q(x_1, x_2) = q_1(x_1) \ q_2(x_2)$, which minimizes $KL(p || q)$.

Let us first look at the joint distribution: $p(x_1, x_2) = p(x_1) \ p(x_2 | x_1)$

Now the Kullback-Leibler divergence for these dependent Bernoulli distributed random variables: $KL(p || q) = \sum_{(x_1, x_2) \in \{0, 1\}^2} p(x_1) \ p(x_2 | x_1) \log \frac{q_1(x_1) \ q_2(x_2)}{p(x_1) \ p(x_2 | x_1)}$

$KL(p || q) = \sum_{(x_1, x_2) \in \{0, 1\}^2} p(x_1) \ p(x_2 | x_1) [\log (q_1(x_1) + \log q_2(x_2) - \log p(x_1) - \log p(x_2 | x_1)]$

I will now document all of the four cases in a table to ease the computations a little bit.

$ \begin{array}{c|c|c|c|c|c} x_1 & x_2 & p(x_1) & p(x_2 | x_1) & q(x_1, x_2) & p(x_1) \ p(x_2 | x_1) \left[ \log q_1(x_1) + \log q_2(x_2) - \log p(x_1) - \log p(x_2 | x_1) \right] \\ \hline 0 & 0 & 0.4 & 0.5 & q_1(0) q_2(0) & 0.4 \cdot 0.5 \left[ \log q_1(0) + \log q_2(0) - \log 0.4 - \log 0.5 \right] \\ 0 & 1 & 0.4 & 0.5 & q_1(0) (1 - q_2(0)) & 0.4 \cdot 0.5 \left[ \log q_1(0) + \log (1-q_2(0)) - \log 0.4 - \log 0.5 \right] \\ 1 & 0 & 0.6 & 0.9 & (1 - q_1(0)) q_2(0) & 0.6 \cdot 0.9 \left[ \log (1-q_1(0)) + \log q_2(0) - \log 0.6 - \log 0.9 \right] \\ 1 & 1 & 0.6 & 0.1 & (1- q_1(0)) (1 - q_2(0)) & 0.6 \cdot 0.1 \left[ \log (1-q_1(0)) + \log (1-q_2(0)) - \log 0.6 - \log 0.1 \right] \\ \end{array} $

Denote $y = (y_1, y_2) = (q_1(0), q_2(0))$. Now using that table we can write $KL(p||q)$ in a bit of a simpler form to make the oncoming gradient computation a bit easier.

$KL(p||q) = 0.4 \log y_1 + 0.6 \log (1-y_1) + 0.74 \log y_2 + 0.26 \log(1-y_2) + \text{const} $

Now we can compute it's gradient w.r.t. to $y_1$ and $y_2$:

$\Rightarrow \nabla_{y_1, y_2} KL(p||q) = \begin{bmatrix} 0.4 \frac{1}{y_1} - 0.6 \frac{1}{1-y_1} \\ 0.74 \frac{1}{y_2} - 0.26 \frac{1}{1-y_2} \end{bmatrix}$

Setting that to zero we get the following system of equations:

$\begin{cases} 0.4 \frac{1}{y_1} - 0.6 \frac{1}{1-y_1} &=0 \\ 0.74 \frac{1}{y_2} - 0.26 \frac{1}{1-y_2} &=0 \end{cases}$

$0.4 \frac{1}{y_1} - 0.6 \frac{1}{1-y_1} = 0 \Leftrightarrow 4 (1-y_1) - 6 y_1 = 0 \Rightarrow y_1 = 0.4$

$0.74 \frac{1}{y_2} - 0.26 \frac{1}{1-y_2} = 0 \Leftrightarrow 74 (1-y_2) - 26 y_2 = 0 \Rightarrow y_2 = 0.74$

Since the logarithm is convex and therefore linear combination (with positive coefficients) of logarithms is also convex. Therefore the gradient's zero must be a global minimizer.

Therefore to minimize the Kullback-Leibler divergence we set: $q_1(x_1=0) = 0.4 = 1-q_1(x_1=1)$ and $q_2(x_2=0)=0.74 = 1-q_2(x_2=1)$

And with those the approximator becomes: $q(x_1, x_2) = q_1(x_1) q_2(x_2)$ with the Bernoulli distributions defined above.
